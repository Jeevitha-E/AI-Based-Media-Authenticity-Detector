#  AI Media Authenticity Detector

## Overview

An AI-based web application that detects whether images, audio, and videos are real or fake.
The system uses deep learning models to identify manipulated media and provide a confidence score.

-----------------------------

## 🚀 Features

* Image deepfake detection
* Audio authenticity detection
* Video frame analysis
* Confidence score output
* Simple web interface (Streamlit)

-----------------------------

## 🛠️ Tech Stack

* Python
* Streamlit
* TensorFlow / Keras
* PyTorch
* OpenCV, Librosa, YOLO
